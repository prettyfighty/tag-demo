詳細步驟與說明，請參考[這篇文章](https://lailai-blog.gq/2021/07/16/Tag-With-Select2)

# Version

- Ruby 2.7.2

- Rails 6.1.4

# Install

1. $`bundle install`

2. $`yarn install`

3. $`rails db:migrate`

4. $`rails db:seed`

5. $`foreman s`

之後打開瀏覽器，輸入網址：**localhost:3000**
